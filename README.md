# PRODIGY_ML_01
A machine learning project that predicts house prices using Linear Regression based on area, bedrooms, and bathrooms. Built using Python in a mobile-friendly environment (Google Colab) with data visualization and real-time prediction capability.
# 🏠 House Price Prediction using Linear Regression

## 📌 Project Description
This project predicts house prices based on:
- Area
- Number of bedrooms
- Number of bathrooms

It uses Linear Regression from Scikit-learn.

## 🚀 Features
- Simple and fast model
- User input prediction
- Data visualization
- Mobile-friendly (runs on Google Colab)

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## 📊 How It Works
1. Dataset is created or loaded
2. Model is trained using Linear Regression
3. User inputs values
4. Model predicts house price

## 📱 Platform
This project is built and executed using Google Colab (mobile-friendly).
